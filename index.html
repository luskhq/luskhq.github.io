Built at lusk

  • arturmuller/lobster-chat

    Full-stack experiments by the Lusk team.

    This is our tech playground. We use it to test drive most the shiny new things we want to use in Lusk. Think about it as a step above Todo MVC. Currently using React, Redux, Koa, Socket.io, Mongoose, Webpack...

  • arturmuller/select-layers-by-name

    Sketch Plugin for creating a selection using layer names

    When working with large Sketch files with a lot of groups and nesting, selecting them by hand in the Layer List is a massive pain. This plugin uses layer names for creating selections which can significantly speed up your workflow.

  • pavelvolek/redux-keen

    Redux middleware for sending analytics to Keen

    We use (and love!) Keen.io Analytics. Really, it's awesome. This middleware makes it really easy to get up and running using Keen in your Redux app.

  • saamo/api-blueprint-focus-booster

    Gulp tasks for writing large API Blueprints efficiently

    API Bluprints are pretty great. We use them all the time to synchronise our front-end/back-end development. This group of Gulp tasks make writing large API Blueprints a breeze.

  • saamo/api-doc-test

    Generate API Blueprint from BDD-style (Jasmine/Mocha) test

    Writing documentation takes time. Writing tests takes time. Using this package on your tests, you can significantly speed up your whole process, effectively only having to write and you get your docs for free! Yay!
